C language

C is a fast, portable programming language known for its efficiency and close-to-the-hardware functionality, widely used in system programming and embedded systems.

C++

C++ is a powerful, efficient programming language renowned for its flexibility and support for both procedural and object-oriented programming paradigms

Python

Python is a versatile, readable language prized for its simplicity and vast library ecosystem, popular across web development, data analysis, and AI.

Java

Java is a widely-used, object-oriented programming language known for platform independence and used for enterprise applications, mobile apps, and web services.

R programming

R is an open-source language for statistical computing and data analysis, featuring extensive libraries for machine learning and visualization, widely used in academia and industry.
